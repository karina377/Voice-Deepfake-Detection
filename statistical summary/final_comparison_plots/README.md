# Final Comparison Plots

## Overview

This folder contains graphical visualizations of the statistical evaluation results obtained during the model comparison phase.

The figures presented here are visual representations of the summary tables located in the **evaluation_tables** folder. Their purpose is to simplify the interpretation of the experimental results and provide an intuitive comparison between the evaluated deep learning models.

The plots summarize the most important performance and efficiency metrics that were used to select the final deployment model.

---

## Figures Description

### Figure 2.1 – Accuracy Comparison Between Models

Displays the overall classification accuracy achieved by each evaluated model.

This figure provides a quick comparison of the overall detection performance across all models.

---

### Figure 2.2 – Correct and Misclassified Audio Files

Illustrates the percentage of correctly classified and misclassified recordings for each model.

The figure visually emphasizes the error rate of each model and highlights the differences in classification reliability.

---

### Figure 2.3 – Classification Metrics Comparison Between Models

Compares the primary evaluation metrics for all models, including:

- Precision
- Recall
- F1-Score
- ROC-AUC

This figure enables a comprehensive comparison of classification quality beyond overall accuracy.

---

### Figure 2.4 – Average Inference Time per Audio File

Compares the average inference time required to classify a single audio recording.

This figure highlights the computational efficiency of each model and supports comparisons between detection accuracy and inference speed.

---

### Figure 2.5 – Number of Parameters Comparison

Displays the total number of trainable parameters for each model.

The figure illustrates differences in model complexity and helps explain variations in computational requirements and deployment suitability.

---

## Purpose

These visualizations complement the statistical tables by presenting the evaluation results in an intuitive graphical format.

Together with the tables in the **evaluation_tables** folder, they provide a comprehensive overview of:

- Overall classification performance
- Error distribution
- Classification quality metrics
- Computational efficiency
- Model complexity

These figures were also used throughout the project report and final presentation to support the model comparison and final model selection.
