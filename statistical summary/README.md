# Statistical Summary

## Overview

This folder contains all statistical analyses, summary tables, visualizations, and notebooks used to evaluate and compare the deep learning models developed in this project.

The analyses were generated from the prediction results of all evaluated models after testing on the final dataset. They were used to compare classification performance, computational efficiency, recording-level behavior, and to support the final model selection.

The folder is organized into several subfolders, each containing a different type of analysis.

---

## Folder Structure

### evaluation_tables/

Contains the main summary tables generated from the evaluation process, including:

- Training configuration
- Classification metrics
- Final model ranking

These tables provide the quantitative results used to compare the models.

---

### final_comparison_plots/

Contains graphical visualizations of the evaluation tables.

The figures present the statistical results in an intuitive visual format, making it easier to compare model performance, classification metrics, computational efficiency, and model complexity.

---

### model_recording_analysis/

Contains detailed recording-level analyses.

These files investigate how each model performed on every recording, identify common classification errors, compare performance across spoofing attack types, analyze difficult recordings, and summarize computational efficiency.

---

## Main Notebooks

### eda.ipynb

Contains the exploratory data analysis (EDA) performed on the evaluation results, including statistical summaries and visual inspection of model performance.

---

### final_model_comparison.ipynb

The main notebook responsible for generating all statistical analyses included in this folder.

It produces:
- Summary evaluation tables
- Recording-level analyses
- Performance comparison statistics
- Computational efficiency analysis
- Final comparison figures

The outputs generated by this notebook are organized into the corresponding subfolders described above.

---

## Purpose

This folder provides the complete statistical evaluation of the proposed deep learning models.

Together, the notebooks, tables, and visualizations provide a comprehensive comparison of model performance and support the final decision to select the deployment model.
